# PokerHandsProject
Poker Hands Sorter


Required Setup

    -Build with open jdk-16.0.2
    -Clone repo


Method 1 - 
Run in Application

    -Open project in preferred IDE.
    -Run App.main


Method 2 - 
Open in Terminal

    -Navigate to directory
    -execute: 
    java -jar PokerHands.jar src/test/data/poker-hands.txt
